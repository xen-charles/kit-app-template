# XENOPS Pendulum Demo

A physics-based swinging pendulum demonstration for Omniverse Kit.

## Features

- Physics-based pendulum simulation using PhysX
- Realistic gravity and momentum
- Visual pendulum bob and rod
- Easy to create through the menu system

## Usage

1. Launch the Pendulum Demo application
2. Go to Create > Physics > Pendulum Demo
3. Watch the pendulum swing!

## Technical Details

- Uses USD Physics for realistic simulation
- Revolute joint connects the pendulum bob to anchor point
- Physics scene with gravity simulation
- Proper mass and collision properties
